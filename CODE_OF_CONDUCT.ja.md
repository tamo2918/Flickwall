# 行動規範

[English](CODE_OF_CONDUCT.md) | 日本語

Flickwall は、敬意があり、実用的なオープンソースプロジェクトであることを目指します。

## 期待される振る舞い

- 率直かつ建設的に伝える。
- 批判はコード、設計、挙動、保守性に向ける。
- 善意を前提にしつつ、根拠と明確さを大切にする。
- 経験レベルや開発環境の違いを尊重する。

## 許容されない振る舞い

- ハラスメント、脅迫、個人攻撃
- 差別的な言葉や行動
- 許可なく個人情報を公開すること
- 繰り返しの無関係な妨害

## 運用

メンテナーは、プロジェクトをコントリビューターやユーザーにとって敵対的または危険な
場所にするコメント、Issue、ユーザー、コントリビューションに対して、削除、クローズ、
ブロック、または受け入れ拒否を行う場合があります。
