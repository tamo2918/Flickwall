# 変更履歴

軽量な変更履歴として管理します。

## 未リリース

- まだありません。

## 0.1.1 - 2026-05-22

- アプリアイコンのアセットを追加しました。
- README のヘッダーにアプリアイコンを追加しました。
- 追加したフォルダを継続的な同期元として保存するようにしました。
- フォルダ内の画像追加、リネーム、編集、削除がライブラリに反映されるようにしました。
- 起動時にライブラリウィンドウを自動表示せず、メニューバー項目から開く挙動に修正しました。
- ライブラリウィンドウを開いた状態でも、ショートカットキーを離した時に選択中の壁紙が適用されるように修正しました。

## 0.1.0 - 2026-05-22

- ネイティブ macOS の壁紙ライブラリとスイッチャーを追加しました。
- カスタマイズ可能なグローバルショートカットを追加しました。
- セキュリティスコープ付きブックマークを使った、サンドボックス対応の画像・フォルダインポートを追加しました。
- 非同期サムネイル生成とメモリ内サムネイルキャッシュを追加しました。
- 壁紙メタデータの保存先を `UserDefaults` から Application Support の JSON に移行しました。
- 壁紙カードの相対的な「最後に適用」時刻ラベルを削除しました。
- 英語・日本語のドキュメントを追加しました。
- README 用のデモ GIF を追加しました。
- 初回の GitHub Release 配布物を追加しました。
