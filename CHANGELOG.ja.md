# 変更履歴

[English](CHANGELOG.md) | 日本語

正式なリリースを開始するまでは、軽量な変更履歴として管理します。

## 未リリース

- ネイティブ macOS の壁紙ライブラリとスイッチャーを追加しました。
- カスタマイズ可能なグローバルショートカットを追加しました。
- セキュリティスコープ付きブックマークを使った、サンドボックス対応の画像・フォルダインポートを追加しました。
- 非同期サムネイル生成とメモリ内サムネイルキャッシュを追加しました。
- 壁紙メタデータの保存先を `UserDefaults` から Application Support の JSON に移行しました。
- 壁紙カードの相対的な「最後に適用」時刻ラベルを削除しました。
