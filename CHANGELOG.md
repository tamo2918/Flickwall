## Unreleased

- Nothing yet.

## 0.1.1 - 2026-05-22

- Added app icon assets.
- Added the app icon to the README header.
- Added persistent folder sources so imported folders stay connected to Flickwall.
- Added live folder syncing for added, renamed, edited, and removed image files.
- Fixed app launch so the library window stays hidden until opened from the menu bar item.
- Fixed shortcut release application when the library window is already open.

## 0.1.0 - 2026-05-22

- Added the initial native macOS wallpaper library and switcher.
- Added customizable global shortcuts.
- Added sandbox-safe image and folder import with security-scoped bookmarks.
- Added asynchronous thumbnail generation and in-memory thumbnail caching.
- Moved wallpaper metadata storage from `UserDefaults` to Application Support JSON.
- Removed relative "last applied" time labels from wallpaper cards.
- Added bilingual English/Japanese documentation.
- Added a README demo GIF.
- Added the first GitHub Release distributable.
